# RegenTribe Solution Archive

Based on [The Hive](https://tribesplatform.app/hive) categories.

## Structure

```
archive/
├── 01-ecology/          # Earth Systems, Science & Agriculture
├── 02-hardware/         # Built Environment & Infrastructure
├── 03-humanware/         # Human Systems, Culture, Wellbeing
├── 04-economy/           # Regenerative Finance & Exchange
└── 05-technology/        # Software, Intelligence, Digital Systems
```

## Adding Solutions

1. Find the matching category + subcategory folder
2. Add a markdown file with:
   - Name & brief description
   - Source/link
   - Pros & cons
   - Cost range
   - DIY vs purchased
   - Best for (location, scale, etc.)
   - Any relevant tags
