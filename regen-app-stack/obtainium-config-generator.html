What is Obtainium?

Obtainium is an open-source, permissions-minimal APK manager for Android. It lets you install apps directly from GitHub releases — no App Store, no middleman.

How to use this generator:

  • Enter a GitHub repo URL that publishes Android APKs in its releases
  • The generator creates a URL or JSON config for Obtainium
  • In Obtainium, tap + → Add from Clipboard (for URL) or Add from QR
  • Or paste the JSON via Add from Clipboard if Obtainium supports it

Supported URL formats:

  • https://github.com/author/repo
  • https://github.com/author/repo/releases
  • https://github.com/author/repo/releases/latest

How Obtainium finds APKs

Obtainium scrapes the GitHub releases page and finds any .apk files attached to the latest release. It then:

  • Downloads and verifies the APK
  • Checks for updates by comparing versions
  • Notifies you when updates are available

You verify the APK signature manually (or trust the GitHub build pipeline).
