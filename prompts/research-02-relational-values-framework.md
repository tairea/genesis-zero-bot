# Research Prompt 02: Relational and Indigenous Values Framework

## Context

LJMap measures 130 individual-level values through ranking. The values are Western, individualist, agent-centric. Indigenous frameworks treat values as relational — they exist between entities, not within individuals. This creates both a cultural bias problem and a conceptual gap.

## Research Questions

### Core Problem
How do you construct a values framework that is not culturally bounded to Western individualism, and that treats values as relational rather than as personal inventory?

### Specific Investigations

**Q1: What does "values as relational" actually mean, operationally?**
- Literature: Indigenous psychologies (M. K. Deo, Maputake L. T. Durie frameworks), Ubuntu philosophy, Gift culture (Mauss, Hyde), Indigenous research methodology (Linda Tuhiwai Smith)
- Find: NOT a list of indigenous values — a structural description of how relational values differ from individual values
- Distill: The grammar of relational values. How do you MEASURE something that exists between people rather than inside them?

**Q2: What is the relationship between land, ancestors, and community as value-holders?**
- Indigenous worldviews: land is not resource, it is relative. Ancestors are present. Community includes the dead and the unborn.
- Research: bioregionalism (Sale), deep ecology (Naess), indigenous land tenure, Ubuntu ontology
- Find: A VALUES vocabulary for human-land-ancestor-community that does not reduce to "environmentalism" or "collectivism"
- Propose: A set of relational value categories that could be used in a community design context

**Q3: What is the difference between "extractive" and "regenerative" value vocabularies?**
- LJMap includes: Profit, Owning, Competition, Efficiency — these are extractive-colonial vocabulary dressed as neutral values
- Research: degrowth economics (Latouche), Doughnut Economics (Raworth), Buen Vivir, Ubuntu economics
- Find: A principled distinction between values that assume growth/extraction and values that assume regeneration/reciprocity
- Deliverable: A VALUE WORD AUDIT framework — how to interrogate any values list for extractive assumptions

**Q4: How do you translate across cultural value frameworks without flattening?**
- LJMap ontology vs. Indigenous relational ontology vs. Permaculture ethics — these are not equivalent, not interchangeable
- Research: cross-cultural psychology (etic vs. emic), translation theory (Eco), indigenous data sovereignty
- Find: A method for identifying equivalences AND irreducible differences between value frameworks
- Deliverable: A TRANSLATION MATRIX template — structure for mapping one ontology to another while preserving what doesn't translate

**Q5: Can a values framework include what cannot be said?**
- LJMap: every value must be a word that can be ranked. But sacred things resist wordification
- Research: apophatic theology, negative capability (Keats), the sacred as undefinable, Dark Mountain Project (Kingsnorth)
- Find: A place in the framework for the unsayable — what cannot be measured because measuring it destroys it
- Deliverable: A PROTOCOL FOR THE SACRED — how to acknowledge what cannot be formalized without formalization collapsing

## Deliverables

1. **Relational Values Grammar** — The structural description of how relational values differ from individual values, with measurement implications
2. **Land-Ancestor-Community Value Vocabulary** — A set of relational value categories (not a list, a STRUCTURE) for regenerative community design
3. **Value Word Audit Framework** — A method for interrogating any values list for extractive/colonial assumptions
4. **Cross-Framework Translation Matrix** — Template and methodology for mapping across cultural value systems
5. **Sacred Protocol** — How to hold space for what cannot be formalized in a values framework

## Execution Instructions

Work with primary sources — go to the thinkers named, not just secondary summaries. Indigenous knowledge requires respectful distance; do not synthesize faster than the source material allows. When in doubt, note what you don't know rather than filling the gap.
