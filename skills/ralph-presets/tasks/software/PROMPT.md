# SOFTWARE - Digital Infrastructure

## Overview
Technology systems including AI, connectivity, data management, and digital tools for regenerative communities.

## Categories
- **Connectivity** - Internet, mesh networks
- **Software Applications** - Governance, management tools
- **Data** - Databases, GIS mapping
- **LLMs & AI** - Assistants, decision support
- **Robotics** - Automation, farming bots
- **IoT & Sensors** - Environmental monitoring
- **3D Mapping** - Site analysis, planning
- **Blockchain** - Governance, transparency
- **Gamification** - Engagement systems

## Key Technologies
- Meshtastic for offline communication
- Radicle for version control
- TerminusDB for knowledge graphs
- OpenCropper for mapping
- Various AI assistants
