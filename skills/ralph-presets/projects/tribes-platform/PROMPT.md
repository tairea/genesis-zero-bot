# Tribes Platform Research & Development

## Objective
Research and implement improvements to the Tribes Platform (WordPress-based community platform).

## Current State
- WordPress tribes platform running at tribesPlatform.app / tribes.regentribe.org
- User onboarding issues identified (profile form confusion)
- Platform bugs being tracked
- V2 planning in progress

## Key Issues to Address

### User Onboarding
1. Profile form fields unclear (magic/passion/purpose)
2. Missing subheaders explaining what fields mean
3. Add estimated time to complete forms
4. Auto-advance to next section after completion

### Technical
1. Website rendering issues (cached CSS)
2. Welcome dashboard page for new users
3. Feature mapping for V2

## Research Required
- No-code platform alternatives
- Similar community platforms
- User experience best practices
