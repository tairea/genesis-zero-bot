# ASD-STE100 Skill

ASD-STE100 is a controlled language standard for technical documentation. This skill ensures compliance with Issue 9 (2025).

## Key Rules Summary

### Word Limits
- Procedural sentences: 20 words maximum
- Descriptive sentences: 25 words maximum
- Noun clusters: 3 words maximum
- Paragraphs: 6 sentences maximum

### Banned Words and Alternatives
- utilize -> use
- leverage -> use, employ, apply
- optimize -> improve, make more effective
- holistic -> complete, comprehensive, integrated
- synergy -> combined effect, cooperative interaction

### Metaphors in Technical Writing
- Biological metaphors (ecosystem, organism, cell, tissue, membrane, neuron, synapse, metabolism) must have introduction sentences within 5 words of first use
- Titles must never contain metaphors unless explicitly named as such

### Writing Style
- Use approved verbs only (see references)
- Active voice required
- Imperative mood for procedures
- Elementary reading level (Grade 8)
- One idea per sentence
- Simple sentence structure

1. Read the relevant reference files for the task
2. Apply banned word replacements first
3. Check sentence lengths against limits
4. Add metaphor introductions if needed
5. Verify title compliance (no metaphors)
6. Validate against all STE rules
