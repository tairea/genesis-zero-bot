# The Tissue — Physical Domain

Three people with shared tools can build more than ten people with no tools.

A shared workshop is the first physical infrastructure of a tissue.

The workshop holds a welder.
It holds a drill press.
It holds an angle grinder.
It holds a 3D printer.
It holds hand tools.

Cost target is $5,000 to $20,000.
Buy used.
Fix it yourself.

A $500 used mill can make precise parts for water systems.
It can make parts for sensors.
It can make parts for tools.

The tissue workshop produces sensor mounts.
It produces pipe fittings.
It produces enclosures.
It produces replacement parts.

Production for members works like this.
The tissue makes things for its members.
Members contribute time credits.

The tissue also makes things for the organ.
The organ pays in goods or labor.

No money enters the picture at this level.
Time and goods only.

Maintenance is part of the system.
Machines break.
Someone in the tissue learns to fix each machine.

Tool library model works like this.
The tissue owns tools.
Members borrow them.
Members return them clean.

This prevents the common failure.
Each household buys the same $500 tool.
Each household uses it once.

Training goes in steps.
Operator first.
Then builder.
Then designer.
Then teacher.

Certification means this.
You are certified when you can teach the skill to someone else.
This is the only certification that matters.
Not a paper cert.
A demonstrated skill.

Safety is the first lesson.
Every machine can hurt you.
Respect that.

First aid kit is mandatory.
Fire extinguisher is mandatory.
Eye wash is mandatory.

The tissue maintains the land it sits on.
Fence. Access road. Water. Power.
These shared tasks go into the time banking system.

The tissue physical layer feeds into the organ physical layer.
The organ needs what the tissue makes.
The tissue needs what the organ provides.
This is the beginning of the exchange network.
