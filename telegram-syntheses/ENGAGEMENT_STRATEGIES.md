# ENGAGEMENT PLAYBOOK

## Phase 1: Assessment
1. Identify actor
2. Map capabilities/vulnerabilities
3. Find leverage points
4. Prepare approach

## Phase 2: Outreach
1. Back-channel first
2. Find internal advocate
3. Propose meeting
4. Share economics

## Phase 3: Negotiation
1. Establish mutual benefit
2. Define terms
3. Test with pilot
4. Scale if successful

## Phase 4: Resolution
1. Formalize agreement
2. Monitor compliance
3. Adjust as needed
4. Document lessons
